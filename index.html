Ghost OS

Your AI agent can write code, run tests, search files. But it can't click a button, send an email, or fill out a form. Ghost OS changes that.

Get Started View on GitHub

Why Ghost OS?

Other computer-use tools take screenshots and guess what's on screen. Ghost OS reads the macOS accessibility tree.

🧠

Self-Learning

A frontier model figures out the workflow once. A small model runs it forever. Show Ghost OS how to do something once, and it remembers forever.

📖

Transparent

Recipes are JSON. Read every step before running. No black box. Know exactly what your AI is doing on your Mac.

🖥️

Native

Accessibility tree first. Vision fallback when needed. Structured data over pixel guessing. Works with any macOS app.

🚀

Any App

Not just browsers. Slack, Finder, Messages — anything on your Mac. Native macOS apps with full functionality.

🔒

Local

Your data never leaves your machine. Run locally with a local vision model. Privacy-first design.

🔌

Open

MCP protocol. Works with Claude Code, Cursor, VS Code, or any MCP client. Open source under MIT license.

29 Tools

Everything your AI agent needs to see and operate your Mac.

🔍 Perception

ghost_context Get current app, window, URL, focused element, and all interactive elements
ghost_state List every running app with windows, positions, and sizes
ghost_find Search for elements by name, role, DOM id, or CSS class
ghost_read Extract text content from any app
ghost_inspect Get complete metadata for one element
ghost_element_at Identify what element is at a specific screen coordinate

📸 Vision

ghost_screenshot Capture a window screenshot for visual debugging
ghost_annotate Screenshot with numbered labels on interactive elements
ghost_ground Find element coordinates using vision (ShowUI-2B)
ghost_parse_screen Detect all interactive elements via vision

🎯 Actions

ghost_click Click an element by name, DOM id, or coordinates
ghost_hover Move cursor to trigger tooltips and hover effects
ghost_long_press Press and hold for context menus and drag initiation
ghost_drag Drag from one point to another
ghost_scroll Scroll up, down, left, or right in any app

⌨️ Input

ghost_type Type text into a specific field
ghost_press Press a single key like Return, Tab, Escape
ghost_hotkey Press key combinations like Cmd+L, Cmd+Return

🪟 Windows

ghost_focus Bring any app or specific window to the front
ghost_window Minimize, maximize, close, move, or resize windows
ghost_wait Wait for URL, element, or title changes

🎓 Recipes

ghost_recipes List all installed recipes with descriptions
ghost_run Execute a recipe with parameter substitution
ghost_recipe_show View full steps and configuration of a recipe
ghost_learn_* Start/stop/status learning mode for workflow capture

Get Started

Install Ghost OS in under a minute.

1

Install via Homebrew

Run the following command in your terminal:

brew install ghostwright/ghost-os/ghost-os
2

Run Setup

This handles permissions, MCP configuration, and vision model setup:

ghost setup
3

That's It!

Your AI agent can now see and operate every app on your Mac. Works with Claude Code, Cursor, VS Code, or any MCP client.

Requires macOS 14+ and Swift 6.2+
